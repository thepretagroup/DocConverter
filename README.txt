
To install:
1. Download NCIDocConverterDeploy_[latest version].zip file to your computer
2. Extract files
3. Execute setup.exe in the Deploy directory
    - You may need to accept the lack of a valid certificate

To Run:
0. Clicking the logo pops up instructions for NCI and deployment version number
1. Click 'Select Input File' button & choose a test file
    - This directory will be saved in the registry for next time
2. Click 'Select Output File' and confirm directory & name
3. Click 'Convert!' to translate the file
4. Click 'View/Edit Output' to validate output
5. Repeat steps 1-4
6. Click 'Exit' to end application

To Uninstall:
1. Remove app from the Windows Apps & Features control panel
